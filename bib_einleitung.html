Java Entwicklungsumgebung

Um die Rätsel gemeinsam mit Botchi lösen zu können, benötigst du im Laufe der Zeit eine Java-Entwicklungsumgebung.
Eine Entwicklungsumgebung ist eine Software-Anwendung, die Entwicklern hilft, Java-Programme effizient zu schreiben, zu testen und zu debuggen.

Für dieses Spiel wird die Entwicklungsumgebung IntelliJ IDEA von JetBrains empfohlen, Du kannst aber auch eine Entwicklungsumgebung Deiner Wahl nutzen.

Eine Java Entwicklungsumgebung – Integrated Development Environment, IDE besteht aus folgenden Hauptkomponenten und Funktionen:

  • Quellcode-Editor
    • Ein Texteditor für das Schreiben von Code
    • Bietet Funktionen wie Syntaxhervorhebung, automatische Vervollständigung und Fehlererkennung
  • Compiler
    • Übersetzt den Quellcode (Java-Dateien) in Bytecode (class-Dateien), den die Java Virtual Machine (JVM) ausführen kann
  • Debugger
    • Hilft Entwicklern, den Fehler im Code zu finden und zu beheben
  • Versionskontrolle
    • Integration von Versionskontrollsystemen wie Git
    • Erlaubt das Verfolgen von Änderungen im Code und das Zusammenarbeiten mit anderen Entwicklern

Um IntelliJ IDEA vollständig nutzen zu können muss zunächst das OpenJDK (Open Java Development Kit) auf Deinem Rechner installiert werden.

Installationsanleitung OpenJDK:

  1. Verzeichnis Java unter C:\Programme erstellen
  2. Neueste Version von: https://jdk.java.net herunterladen und ZIP-Datei in das zuvor Erstellte Verzeichnis entpacken
  3. Umgebungsvariable Path hinzufügen
    • In das Erstellte Verzeichnis wechseln
    • Verzeichnis bin öffnen
    • Pfad des Verzeichnisses in der Kopfzeile kopieren
  4. Den Begriff „Umgebungsvariable“ im Suchfeld der Taskleiste eingeben
    • „Umgebungsvariable Für dieses Konto“ auswählen
    • Path auswählen und bearbeiten
    • Neu – Zuvor kopierten Pfad einfügen
    • Ok – Speichern
  5. Terminal öffnen und Java Version abfragen
    • Eingabe: javac -version

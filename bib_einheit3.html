Lerneinheit 3 - Eingabemöglichkeiten in Java

Diese Lerneinheit hat das Ziel, Dir den Umgang mit externen Eingabedaten zu vermitteln. Dabei sollst Du ein Verständnis für die Handhabung unvorhersehbarer Daten entwickeln. Darüber hinaus sollst Du den Umgang mit verschiedenen Datentypen bei der Eingabe erlernen. Du sollst ein Gespür dafür entwickeln, Deinen Code so zu gestalten, dass eine effiziente Weiterverarbeitung der Daten möglich ist.

Lerninhalte

  • Eingaben (Konsole, Scanner)
  • Was ist eine GUI?
  • Daten weiterverarbeiten
  • String Addition
  • Parsen
  • String Verarbeitung
